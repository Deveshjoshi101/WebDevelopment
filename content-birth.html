The Birth of the Web (Early 1990s)

1989-1990: Tim Berners-Lee created the first web browser, called "WorldWideWeb" (later renamed Nexus), and the first web server software.

1991: The first webpage, which provided information on the World Wide Web project, went live. It explained what the web was and how to use it.

1993: CERN released the source code for the WorldWideWeb browser, making it freely available to the public. This was a crucial step in democratizing the web.

1993: The National Center for Supercomputing Applications (NCSA) released Mosaic, the first widely-used graphical web browser, which greatly contributed to the web's popularity.
